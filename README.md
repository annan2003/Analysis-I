# Analysis-I
Analysis I (LMU Informatik &amp; Statistic)

**Welcome to the Mathematics Analysis I course for the Winter Semester 2025/26**
This repository provides supplementary learning materials for students enrolled in Informatik and Statistik programs.

**Weekly Uploads:**
On the weekend, I will upload the my script from the previous week.
These scripts are for reference only and do not represent the official course materials.
If you notice any mistakes or have suggestions for improvement, feel free to start a discussion or reach out. Constructive feedback is always welcome!

**Exam Preparation:**
Approximately two weeks before the final exam, I will upload past exam papers (Altklausuren) to assist with exam preparation.

- The provided materials are intended solely for personal study use.
